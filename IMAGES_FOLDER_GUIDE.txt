Notes & recommendations:
- If you copy this HTML to another folder or machine, copy the
	entire `images/` folder (including `Video1.mp4`) into the new
	location so all visuals and the welcome video load correctly.
- The page gracefully degrades: missing images display a small
	placeholder created by `handleImgErr()` so layout and
	interactivity continue to work.
- If you prefer a minimal standalone simulator only, you can keep
	just the images used by the simulator (the simulator itself uses
	background layers and a few sprite images if present). For a
	full-course copy, include all referenced files above.
